# 2026-04-06 생존 수당 HUD 추가

## 배경

기존 KillSnail은 달팽이에게 얼마나 오래 살아남았는지에 대한 숫자 보상이 없어, 한 판의 긴장감은 있어도 결과가 또렷하게 남지 않았습니다.

## 목표

- 달팽이가 추격 중일 때 초당 1억 원씩 금액이 올라가게 만든다.
- 잡히면 최종 획득 금액을 `YOU DEAD` 화면에 보여준다.
- 리셋 시 새 판이 0원부터 다시 시작되게 만든다.

## 변경 내용

- `KillSnailCore`에 `earnedMoney` 상태를 추가했습니다.
- `.chasing` 상태에서만 `deltaTime` 기반으로 금액이 누적되게 했습니다.
- `paused` / `dead` 상태에서는 누적이 멈추도록 했습니다.
- 추격 중 상단 HUD에 현재 생존 수당을 표시했습니다.
- 사망 오버레이에 최종 획득 금액 카드를 추가했습니다.

## 설계 이유

- 금액 누적은 UI 효과가 아니라 판정 규칙에 가까워서 코어가 소유해야 합니다.
- AppKit은 이미 존재하는 스냅샷을 받아 렌더링만 하도록 유지했습니다.
- 금액 표시는 한국어 단위(`억`, `만`)와 원화 기호를 섞어 짧고 읽기 쉽게 만들었습니다.

## 영향 범위

- `KillSnailCore` 상태 모델과 업데이트 규칙
- 추격 중 HUD UI
- 사망 오버레이 UI
- 릴리즈 버전 `0.1.2`

## 검증 방법

- `xcodegen generate`
- `swift build`
- GitHub Actions CI / Release
